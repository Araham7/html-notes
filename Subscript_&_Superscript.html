Some basic formulae Algebra :

            

                (1). (a+b)2 = ( a2 + 2ab + b2)
                (2). (a-b)2 = ( a2 - 2ab + b2)
                (3). {(a+b)2 + (a-b)2} = 2(a2 + b2)
                (4). {(a+b)2 - (a-b)2} = 2ab
                (5). (a+b)3 = ( a3 + 3a2b + 3ab2 + b3) = { a3 + 3ab(a+b) + b3 }
                (6). (a-b)3 = ( a3 - 3a2b + 3ab2 - b3) = { a3 - 3ab(a-b) - b3 }
                (7). (a2 - b2) = (a+b).(a-b)
                (8). (a3 + b3) = (a+b).(a2 - ab + b2)
                (9). (a3- b3) = (a-b).(a2 + ab + b2)
